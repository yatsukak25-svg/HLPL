# TutorFlow — Платформа для організації навчання

TutorFlow — це сучасний веб-застосунок на базі ASP.NET Core MVC, розроблений для спрощення взаємодії між репетитором та учнями. Платформа дозволяє керувати розкладом, домашніми завданнями, навчальними матеріалами та фінансовим обліком в одному місці.

## Основний функціонал

### Для репетитора (Tutor):
*   **Керування учнями:** Створення профілів учнів та перегляд їхньої успішності.
*   **Розклад:** Планування занять за допомогою календаря або списку.
*   **Домашні завдання:** Призначення завдань, встановлення дедлайнів та перевірка статусів виконання.
*   **Фінанси:** Автоматичний розрахунок доходу за місяць та відстеження боргів учнів.
*   **Інтеграція з API:** Сповіщення про святкові дні через сервіс `Nager.Date` для зручного планування.

### Для учня (Student):
*   **Особистий кабінет:** Перегляд власного розкладу та актуальних домашніх завдань.
*   **Навчальні матеріали:** Доступ до бази знань, посилань та документів за предметами.
*   **Статус оплати:** Перегляд історії проведених занять та стану заборгованості.
*   **Чернетки:** Можливість зберігати цікаві матеріали в обране через механізм сесій.

### Для адміністратора (Admin):
*   **Керування користувачами:** Створення та редагування акаунтів усіх ролей.
*   **Моніторинг:** Загальний перегляд усіх занять та фінансових транзакцій системи.

## Технологічний стек
*   **Framework:** ASP.NET Core 8.0 MVC
*   **ORM:** Entity Framework Core (SQLite)
*   **Identity:** ASP.NET Core Identity (рольова модель: Admin, Tutor, Student)
*   **Architecture:** Clean Architecture (Web, Application, Domain, Infrastructure)
*   **Patterns:** Generic Repository, Specification Pattern, DTO + Mapping (AutoMapper-style)
*   **UI:** Bootstrap 5, Razor Pages, Partial Views
*   **Інше:** Sessions, Async/Await, External API Integration

## Структура проєкту
*   **TutorFlow (Web):** Контролери, представлення та моделі представлення (ViewModels).
*   **TutorFlow.Application:** Бізнес-логіка, інтерфейси, DTO та специфікації.
*   **TutorFlow.Domain:** Сутності бази даних, перерахування (Enums) та валідація.
*   **TutorFlow.Infrastructure:** Контекст БД, репозиторії, міграції та зовнішні сервіси.
*   **TutorFlow.AccountingLibrary:** Окрема бібліотека для бухгалтерських розрахунків (Monthly Income, Student Debt).

## Швидкий запуск

1.  **Клонування репозиторію:**
    ```bash
    git clone https://github.com/yatsukak25-svg/HLPL.git
    cd HLPL
    ```

2.  **Підготовка бази даних та запуск:**
    ```powershell
    dotnet restore
    dotnet build /m:1
    dotnet run --project .\TutorFlow.csproj
    ```
    *Під час першого запуску система автоматично створить базу даних `app.db`, застосує міграції та заповнить таблиці тестовими даними.*
